How to create Excel file in VBScript



'===============================================================
' Tutorial 03
'
' This tutorial shows how to create an Excel file (1) that has
' multiple sheets (2) in VBScript. The created Excel file is
' empty and the next tutorial shows how to add data into sheets.
'===============================================================
Click here to see the Excel file

WScript.StdOut.WriteLine("Tutorial 03" & vbcrlf & "----------" & vbcrlf)
    
' Create an instance of the class that creates Excel files (1)
Set xls = CreateObject("EasyXLS.ExcelDocument")

' Create two sheets (2)
xls.easy_addWorksheet_2("First tab")
xls.easy_addWorksheet_2("Second tab")

' Create Excel file
WScript.StdOut.WriteLine("Writing file C:\Samples\Tutorial03.xls")
xls.easy_WriteXLSFile ("C:\Samples\Tutorial03.xls")

' Confirm the creation of Excel file
If sError = "" then
    WScript.StdOut.Write(vbcrlf & "File successfully created. Press Enter to exit...")
Else
    WScript.StdOut.Write(vbcrlf & "Error: " & xls.easy_getError())
End If

' Dispose memory
xls.Dispose

WScript.StdIn.ReadLine()
