Getting started with EasyXLS - Excel Library for .NET

Step 1: Download EasyXLS

Download the trial version of EasyXLS Excel Library for .NET from this link:


If you already own a license key, you may login to the account that purchased the license and download EasyXLS:
https://www.easyxls.com/purchased-items

Step 2: Install EasyXLS

Install EasyXLS library.

Step 3: Create a project

Create a project for C#, VB.NET, C++.NET or J# programming language.

Step 4: Add reference to EasyXLS

Add a reference to EasyXLS.dll that can be found under installation path, C:\Program Files\ EU Outsourcing\EasyXLS Trial for .NET vX.x\Dot NET version\DLL by default.

Step 5: Start using EasyXLS

Start using EasyXLS Excel library with C#, VB.NET, C++.NET or J# programming language. Check this User Guide for more samples.
