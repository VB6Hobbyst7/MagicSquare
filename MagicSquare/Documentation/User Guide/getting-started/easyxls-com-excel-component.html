Getting started with EasyXLS - COM+ Excel Component

Step 1: Download EasyXLS

Download the trial version of EasyXLS Excel component for .NET from this link:


If you already own a license key, you may login to the account that purchased the license and download EasyXLS:
https://www.easyxls.com/purchased-items

Step 2: Install EasyXLS

Install EasyXLS library. If EasyXLS component was successfully registered, it should be present under Component Services. If not present or for more details check how to register EasyXLS component.

Step 3: Start using EasyXLS

Start using EasyXLS Excel component with C++, PHP, ASP classic, VB6 or VBS programming language. Check this User Guide for more samples.
