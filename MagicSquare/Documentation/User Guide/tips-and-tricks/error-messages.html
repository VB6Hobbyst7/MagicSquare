How to retrieve, display or hide error messages

Concept

EasyXLS™

library provides methods for handling error messages and warning messages. The error messages can be retrieved for further use in your application and the warning messages can be suppressed from being displayed on the console.

Retrieve error messages

ExcelDocument.easy_getError method returns the error messages that might occur when using EasyXLS library.

Hide error messages and warnings

By default, all the error messages and warnings are displayed on console. By setting to false the flag on ExcelOptions.setDisplayErrors method, this behavior changes.

Related methods:

ExcelDocument.easy_getError
ExcelOptions.setDisplayErrors
