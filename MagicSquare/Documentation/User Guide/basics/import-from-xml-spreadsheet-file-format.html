Import XML Spreadsheet file in .NET, Java and other programming languages

Concept

EasyXLS™

allows you to import data from XML Spreadsheet file format.

An XML Spreadsheet file is an XML file in a format designed by Microsoft specifically for Excel worksheets.

For more technical information about XML spreadsheets you can check XML Spreadsheet Reference web page.

Read XML Spreadsheet file into DataTable in C# and VB.NET

EasyXLS library can be used to import XML Spreadsheet file into DataTable. The DataTable can be the used as data source of a GridView, DataGridView, DataGrid or for any other purposes.

The data from the XML Spreadsheet file can be read using ExcelDocument.easy_ReadXMLSpreadsheet_AsDataSet method and then the DataTable can be obtained from the imported DataSet.
